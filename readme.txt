WordPress page tree widget.

== Description ==

W3 Page Tree Widget is a simple widget plugin for displaying hierarchical page trees. Easily integrates with into any WordPress theme (with widget support). Perfect for online documentation.

== Features ==

* Set widget post type or detect automatically (depending on the page being viewed).
* Widget automatically hidden (non-hierarchical post type, no other pages found, etc.)
* Top-level pages are excluded from the page tree when viewing a sub page.

= Pro Features =

* Expandable and collapsible branches.
* Instantly re-order pages and create new branches. (Functionality restricted to users who can edit other users’ pages–e.g. admins and editors.)
* Set widget post type or detect automatically (depending on the page being viewed).
